## Project Snapshot

Anchor is a biometric grounding companion for people on psychiatric service dog waitlists. Monitors HR/HRV via simulated Garmin data, detects distress triggers, and opens a Claude-powered grounding chat, while logging those previous episodes.

**Stack**: Next.js 16, React 19, Tailwind v4, Supabase, Anthropic SDK, Resend
